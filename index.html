📈 แสดงข้อมูลปีพ.ศ.ที่หน่วยงานได้รับเครื่องมือ

ผลรวมจำนวนเครื่องมือทั้งหมดที่แต่ละหน่วยงานได้รับ ตามข้อมูลที่ตรงกับตัวกรอง

📏 อายุของเครื่องมือ แยกตามหน่วยงาน คลิกแท่งเพื่อกรอง

มากกว่า 16 ปี / ระหว่าง 11-15 ปี / ระหว่าง 1-5 ปี / ระหว่าง 6-10 ปี · แสดงหน่วยงานที่มีเครื่องมือมากที่สุดตามจำนวนที่เลือก

🧪 สัดส่วนเครื่องมือแยกตามประเภท

แยกตาม ด้านน้ำ / อากาศ / ดิน / ฉุกเฉิน

🏛️ หน่วยงานที่มีเครื่องมือ

🔧 ประเภทเครื่องมือที่มีมากที่สุด

รายละเอียดเชิงลึก · ปรับมุมมองกราฟและกรองข้อมูลได้ด้านล่าง

🔋 สถานะการใช้งานตามเครื่องมือ คลิกเพื่อกรอง

สัดส่วนเครื่องมือที่ใช้งานได้ / ชำรุด

📏 แสดงข้อมูลการสอบเทียบของแต่ละหน่วยงาน คลิกแท่งเพื่อกรอง

สอบเทียบแล้ว / ไม่ต้องส่งสอบเทียบ / รอสอบเทียบ / ไม่เคยสอบเทียบ · แสดงหน่วยงานที่มีเครื่องมือมากที่สุดตามจำนวนที่เลือก

🗺️ แผนที่ตำแหน่งหน่วยงานและเครื่องมือ ข้อมูลปรับตามตัวกรองด้านซ้ายอัตโนมัติ

ตำแหน่งจับคู่อัตโนมัติจากคอลัมน์ "หน่วยงาน" ในไฟล์ที่อัปโหลด — คลิกจุด/พื้นที่บนแผนที่ หรือชิปด้านล่าง เพื่อกรองข้อมูลทั้งแดชบอร์ดไปที่หน่วยงานนั้น (เชื่อมกับตัวกรองด้านซ้ายแบบสองทาง)

🧭 เครื่องมือภาคสนาม กรมควบคุมมลพิษ

ข้อมูลชุดเดียวกับแดชบอร์ด — กรองอัตโนมัติตามตัวกรองด้านซ้ายและอัปเดตเมื่อคุณคลิกกราฟ/แผนที่

📚 แหล่งข้อมูล

ใช้ไฟล์เดียวกับที่อัปโหลดไว้ในแดชบอร์างด้านบน — ตำแหน่งจับคู่อัตโนมัติจากคอลัมน์ "หน่วยงาน"

🏷️ รายละเอียดหน่วยงานที่เลือก

ยังไม่ได้เลือกหน่วยงาน
คลิกจังหวัดบนแผนที่ หรือชิปด้านล่างของแผนที่ เพื่อเริ่มต้น
เลือกหน่วยงานก่อน แล้วกดปุ่มนี้เพื่อบันทึก/พิมพ์เป็นไฟล์ PDF ขนาด A4

🗺️ แผนที่เครื่องมือ — หน่วยงานในสังกัดกรมควบคุมมลพิษ

สำนักงานสิ่งแวดล้อมและควบคุมมลพิษที่ 1–16 และหน่วยงานส่วนกลาง — จำนวนเครื่องมือต่อหน่วยงาน ตามตัวกรองปัจจุบัน
